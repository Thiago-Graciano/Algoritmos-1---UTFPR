# Algoritmos 1 - UTFPR 🎓

Repositório dedicado ao armazenamento e organização dos exercícios e projetos desenvolvidos durante as aulas de **Algoritmos 1** na **UTFPR (Universidade Tecnológica Federal do Paraná)**.

O objetivo aqui é praticar a lógica de programação e a sintaxe da linguagem **C**, aplicando os conceitos vistos em sala de aula.

---

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** C
* **Ambiente:** VS Code e Code:Blocks

## 📂 Organização do Repositório

O repositório está dividido em pastas para facilitar a localização dos temas estudados:

### 🚀 Projetos
Sistemas mais completos que integram diversos conceitos aprendidos, focados em resolver problemas específicos.
* Exemplo: `Nota Fiscal Padaria`

### 📝 Listas de Exercícios (Aula)
Resoluções das listas propostas pelos professores durante a disciplina de Algoritmos na UTFPR.
* Exemplo: `If e Else encadeado`

### 🧪 Estudos e Prática Autônoma
Códigos e experimentos criados por iniciativa própria para fixar o conhecimento e testar novas lógicas.
* Exemplo: `Conversão de unidades de medidas`
